# Real-Time Expert Session Booking System

A full-stack real-time expert consultation booking platform built using React, Node.js, Express, MongoDB, and Socket.io.

Users can browse experts, view available slots, book sessions, and track their bookings in real-time.

---

# Live Link:

https://expert-session-booking-system-lyart.vercel.app

---

# Features

## Expert Listing
- View all experts
- Search experts by name
- Filter experts by category
- Pagination support
- Loading and error states

## Expert Details
- View detailed expert information
- View available slots grouped by date
- Real-time slot updates using Socket.io
- Automatically disables booked slots

## Booking System
- Book expert sessions
- Form validation
- Success and error handling
- Prevents double booking

## My Bookings
- Fetch bookings using email
- View booking statuses:
  - Pending
  - Confirmed
  - Completed
